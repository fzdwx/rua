# Configuration Guide

Rua offers extensive customization options to tailor the launcher to your workflow and preferences.

## Configuration Files

The primary configuration file is located at `~/.config/rua/config.json`.

## Basic Configuration

When first launched, Rua creates a default configuration with sensible defaults for appearance, behavior, and window settings.
