# Keyboard Shortcuts

Master Rua's keyboard shortcuts to boost your productivity. This guide covers all available shortcuts and provides tips for efficient usage.

## Global Shortcuts

These shortcuts work system-wide, even when Rua is not visible:

### Primary Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Super + Space` | Toggle Launcher | Show/hide Rua launcher |
| `Super + R` | Show Launcher | Always show launcher (even if visible) |
| `Ctrl + Alt + Space` | Quick Launch | Show launcher with last search |
| `Super + Shift + Space` | Action Mode | Open in action selection mode |

### Alternative Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Alt + F2` | Show Launcher | Alternative launcher shortcut |
| `Ctrl + Alt + R` | Restart Rua | Restart the application |
| `Super + Q` | Quit Rua | Exit the application |

<Callout type="info">
  Global shortcuts can be customized in your desktop environment's keyboard settings or in Rua's configuration file.
</Callout>

## Interface Navigation

When the launcher is open, use these shortcuts to navigate:

### Basic Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `↓` / `Tab` | Next Item | Move to next search result |
| `↑` / `Shift + Tab` | Previous Item | Move to previous search result |
| `Enter` | Execute | Run the selected item |
| `Escape` | Hide | Close the launcher |

### Advanced Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Home` | First Item | Jump to first result |
| `End` | Last Item | Jump to last result |
| `Page Up` | Page Up | Move up by page |
| `Page Down` | Page Down | Move down by page |
| `Ctrl + Tab` | Next Category | Switch to next result category |
| `Ctrl + Shift + Tab` | Previous Category | Switch to previous category |

## Customization

### Custom Shortcuts

You can customize shortcuts in `~/.config/rua/keybindings.json`:

```json
{
  "global": {
    "toggle": ["Super+Space", "Alt+F2"],
    "show": ["Super+R"],
    "hide": ["Escape", "Super+Escape"]
  },
  "interface": {
    "selectNext": ["Down", "Tab", "Ctrl+N"],
    "selectPrevious": ["Up", "Shift+Tab", "Ctrl+P"],
    "execute": ["Return", "Ctrl+O"]
  },
  "custom": {
    "openTerminal": {
      "keys": ["Ctrl+Alt+T"],
      "action": "execute",
      "command": "gnome-terminal"
    }
  }
}
```

### Shortcut Conflicts

If shortcuts conflict with other applications:

1. **Check system shortcuts**: Review your DE's keyboard settings
2. **Modify Rua shortcuts**: Edit the keybindings configuration
3. **Use alternative shortcuts**: Most actions have multiple shortcut options
4. **Disable conflicting shortcuts**: Turn off conflicting system shortcuts

## Usage Tips and Tricks

### Efficient Search Techniques

#### Fuzzy Search

Rua supports fuzzy search - you don't need to type exact names:

- `ff` → Firefox
- `calc` → Calculator  
- `term` → Terminal
- `vsc` → Visual Studio Code

#### Search Operators

Use special operators for advanced searches:

| Operator | Example | Description |
|----------|---------|-------------|
| `>` | `>shutdown` | Execute shell command |
| `?` | `?weather` | Search web |
| `@` | `@john` | Search contacts |
| `#` | `#work` | Search by tag |
| `/` | `/documents` | Search files in path |

#### Quick Calculations

Type mathematical expressions directly:

- `2+2` → Shows result: 4
- `15*8` → Shows result: 120
- `sqrt(16)` → Shows result: 4
- `sin(30)` → Shows result: 0.5

### Workflow Optimization

#### Pinned Items

Pin frequently used items for quick access:

1. Search for an item
2. Press `Shift + Space` to pin
3. Pinned items appear at the top of results

#### Search History

Leverage search history for repeated tasks:

- Use `Ctrl + ↑/↓` to navigate history
- Recent searches appear as suggestions
- Clear history with `Ctrl + H` then `Delete`

#### Action Chaining

Execute multiple actions in sequence:

1. Select multiple items with `Ctrl + Click`
2. Press `Ctrl + Enter` to execute all
3. Or use `Alt + Enter` to choose actions for each

### Power User Features

#### Command Mode

Access advanced features with command mode:

- Type `>` to enter command mode
- Execute shell commands directly
- Access system functions
- Run scripts and automation

#### Extension Workflows

Combine extensions for powerful workflows:

1. **Text Processing**: Select text → Transform → Copy result
2. **File Management**: Search files → Batch operations → Execute
3. **System Control**: Quick settings → Apply changes → Confirm

#### Keyboard-Only Operation

Use Rua entirely without mouse:

1. Launch with global shortcut
2. Type search query
3. Navigate with arrow keys
4. Execute with Enter
5. Access actions with Alt+Enter

## Troubleshooting Shortcuts

### Shortcuts Not Working

1. **Check if Rua is running**:
   ```bash
   ps aux | grep rua
   ```

2. **Verify shortcut registration**:
   ```bash
   rua --list-shortcuts
   ```

3. **Test shortcut conflicts**:
   - Try alternative shortcuts
   - Check system keyboard settings
   - Disable other applications temporarily

### Resetting Shortcuts

```bash
# Reset to default shortcuts
rm ~/.config/rua/keybindings.json
rua --reset-shortcuts

# Or edit manually
nano ~/.config/rua/keybindings.json
```
