# Welcome to Rua

Rua is a modern, extensible command palette launcher for Linux, built with Tauri + React + TypeScript. It provides a fast, lightweight, and highly customizable way to launch applications, execute commands, and interact with your system.

## What Makes Rua Special?

### Modern Architecture
Rua leverages the power of Tauri to provide a native desktop experience while maintaining the flexibility of web technologies. This hybrid approach delivers exceptional performance with a small memory footprint.

### Developer-Friendly
With TypeScript throughout the stack and comprehensive API documentation, Rua makes it easy for developers to create powerful extensions and contribute to the project.

### Extensible by Design
The plugin system allows you to customize Rua to fit your workflow perfectly. From simple shortcuts to complex integrations, extensions can transform how you interact with your system.
